<h1 align="center">YLLREC</h1>
<p align="center">一款轻巧的基于事件的用户轨迹分析插件</p>

## 什么是 YLLREC
> 免责声明: 本产品仅供研究学习, 用途仅仅在于统计分析前端用户使用习惯以及系统故障排查

## 使用方法
```html
<script src="./yllrec.js"></script>
```
